# Сравнение зарплат программистов (HeadHunter vs SuperJob)

Скрипт для сбора и сравнения статистики по вакансиям программистов в Москве с двух крупнейших платформ: HeadHunter и SuperJob.

## Описание

Проект собирает данные по 10 популярным языкам программирования:

- Python, Java, JavaScript, C++, C#, PHP, Go, Ruby, Swift, TypeScript

Для каждого языка считается:

- Общее количество найденных вакансий
- Количество вакансий с указанной зарплатой
- Средняя ожидаемая зарплата (в рублях)

Результат выводится в виде двух красивых таблиц.

## Требования

- Python 3.6 или выше
- Установленные библиотеки:
    - `requests`
    - `python-dotenv`
    - `terminaltables`

## Установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/AydarGataullin7/average_salary.git
```

2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Создайте файл .env в корневой папке и добавьте ваш ключ SuperJob:

```text
API_KEY_SJ=ваш_секретный_ключ
```

## Запуск

```
bash
python work_hh.py
```
